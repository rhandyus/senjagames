🎮 SenjaGames Cart Testing

Test WinPay Virtual Account creation with sample cart items

🚀 Quick Start

Open your browser console (F12) and run:

// Load the testing script import('./cart-test-script.js') // Then use these commands: testPayment() // Quick setup for payment testing listCartItems() // See available test items addRandomToCart(3) // Add 3 random items showCart() // Check cart contents

📋 Available Commands

listCartItems() - List all available test items addToCart('item-id') - Add specific item by ID addRandomToCart(n) - Add n random items to cart showCart() - Show current cart status clearCart() - Clear all items from cart testPayment() - Quick payment test setup

💳 Payment Testing Flow

  1. Open SenjaGames in browser
  2. Open Developer Console (F12)
  3. Run: testPayment()
  4. Go to cart/checkout page
  5. Click "Pay with WinPay VA"
  6. Check console for Virtual Account creation logs

⚠️ Important Notes

  • This script manipulates localStorage directly
  • Items are for testing only - not real accounts
  • WinPay VA creation will use real API calls
  • Make sure your .env has correct WinPay credentials
  • Test on development server first

🔧 Manual Testing

If you prefer manual testing:

// Add items manually window.CartTester.addItem('steam-1') window.CartTester.addItem('epic-1') window.CartTester.showCartStatus() // Or use the quick commands addToCart('battlenet-1') addRandomToCart(2) showCart()
